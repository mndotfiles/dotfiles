## For New Setups!

- Install:
    - Brew 
        - /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    - brew install pip
    - brew install pipenv
    - Python via Pyenv
    - brew install go
    - nvim
    - lunarvim
        - Nerd Font of choice for lunarvim
    - etc etc
- Run this repo `script/setup`
- github
- keyboard settings: CAPSLOCK -> CTRL mapping
